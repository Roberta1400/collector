# Tapped!

## Description

**Tapped!** is a game where players tap falling shapes in 10 seconds to score points. The game features multiple levels. The purpose of the game is to improve reaction time and achive highest scores.

## How to Play

1. Open the game in your browser or mobile device.  
2. Tap **Start** on the Start Screen.  
3. Select a level. Levels can be unlocked by completing previous ones.
4. While playing:
   - Tap on the falling shapes as quickly as possible.
   - Each tapped shape increases your score by 1.  
   - Each level lasts 10 seconds.  
5. After the level ends, the Result Screen will display your score and update your high score if you beat it.  
6. Use **Back to Start Screen** to return or select another level to play.
